# Brackets.py

Содержит функцию check_str, принимающую строку из скобок и проверяющую правильность их расстановки

В реализации использвуется стек по причинам:

 - последняя добавленная скобка должна быть первой закрытой

Стек реализован динамическим массивом:

 - добавление в конец O(1)
 - удаление из конца O(1)
 - доступ по индексу O(1)
